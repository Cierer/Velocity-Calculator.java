The VelocityCalculator program is a console-based Java application used to calculate final velocity using basic physics equations. It follows object-oriented programming (OOP) principles by organizing variables such as initial velocity (vi), acceleration (a), and time (t) with related methods in one class. Users interact with the program via the command line, entering values and selecting between two equations: vf = vi + at or vf² = vi² + 2at.
The program uses the Scanner class from the java.util package to capture user input. It also includes exception handling with a try-catch block to prevent crashes caused by invalid input, ensuring the program continues running by assigning default values.
Its modular structure improves readability and maintenance. The velocityDataEntry() method collects input, velocityEquation() selects the formula, and calculation methods perform computations using the Math class (Math.sqrt() and Math.pow()).
